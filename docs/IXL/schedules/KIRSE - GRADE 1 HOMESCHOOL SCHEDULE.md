## NOTES FOR PARENT

1. **Morning Circle (8:30-8:45):** Start each day with calendar activities - day of week, date, weather, counting days in school.

2. **Breaks:** These are essential! Let Kirse move, play, and have a snack.

3. **IXL Tips:** 
   - Aim for SmartScore of 80+ on each skill
   - Use video tutorials when struggling
   - Games are great rewards after completing skills

4. **Flexibility:** If Kirse is struggling with a concept, spend more time on it. If mastered quickly, move ahead.

5. **Read Aloud:** This is crucial at this age - read to Kirse every day, even if they can read independently.

6. **Celebrate Progress:** Print IXL certificates and celebrate achievements!
